# Telegram Bot attendanceBot

## Описание
Этот проект представляет собой Telegram-бота, который анализирует данные из Excel файла и вычисляет процент выданных и проверенных домашних заданий для каждого преподавателя.

## Установка
1. Склонируйте репозиторий: `git clone <URL репозитория>`
2. Перейдите в директорию проекта: `cd telegram-bot`
3. Установите зависимости: `pip install -r requirements.txt`

## Использование
1. Запустите бота: `python main.py`
2. Отправьте команду `/start` в Telegram, чтобы начать взаимодействие с ботом
3. Отправьте Excel файл с данными, чтобы бот проанализировал их и отправил результаты
